# Weather App ☀️🌧️🌩️

A beautiful, modern Flutter application that provides real-time weather updates for any location worldwide. Instantly check the current weather, forecast, and more with a sleek, intuitive interface.

---

## 🚀 Features

- **Live Weather Data:** Get up-to-date weather information for your city or any location.
- **7-Day Forecast:** Plan ahead with detailed daily forecasts.
- **Beautiful UI:** Clean, minimal, and responsive design.
- **Location Search:** Find weather by city name or use your device's location.
- **Custom Themes:** Light and dark mode support.
- **Weather Icons:** Dynamic icons and backgrounds based on weather conditions.
- **Error Handling:** Friendly messages for network or location issues.

## 🛠️ Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/weather_app.git
   cd weather_app
   ```
2. **Install dependencies:**
   ```bash
   flutter pub get
   ```
3. **Run the app:**
   ```bash
   flutter run
   ```

---

## 🔧 Configuration

- Add your weather API key in the appropriate file (see code comments).
- Customize theme and assets in the `lib/theme` and `assets/` folders.
